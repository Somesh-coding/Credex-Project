# USER INTERVIEWS

## Interview 1 — Student founder

The user understood the savings recommendations quickly but wanted clearer explanations for why certain plans were considered overkill.

Main takeaway:
Recommendation reasoning matters more than raw savings numbers.

## Interview 2 — Small engineering team

The user liked the public share page and said it looked “investor-demo ready.”

Main takeaway:
Visual polish increases trust significantly for finance-oriented products.

## Interview 3 — Developer using multiple AI tools

The user said they often lose track of overlapping subscriptions between ChatGPT, Claude, and coding assistants.

Main takeaway:
Cross-tool visibility is a strong value proposition for this category.
