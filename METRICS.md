# METRICS

North Star metric: qualified audits completed.

This matters because the tool is useful only when a user finishes the audit and sees savings.

Input metrics:
1. Landing page to audit start rate.
2. Audit start to completion rate.
3. Audit completion to email capture rate.

First instrumentation:
- Page view
- Audit started
- Audit completed
- Lead captured
- Consultation clicked

Pivot trigger:
If fewer than 10% of completed audits produce email captures after 100 completed audits, the value proposition or result page needs to change.
